## Описание

Weather Diary - приложение для записи информации о погоде и построения графиков температуры.  
Позволяет добавлять, просматривать и удалять записи о погоде, фильтровать данные и визуализировать изменения температуры.

## Использование

Приложение предоставляет меню с 8 опциями:

1. Показать все записи — список всех записей о погоде
2. Добавить запись — новая запись с валидацией
3. Удалить запись — удаление по номеру
4. Фильтр по дате — поиск записей за конкретную дату
5. Фильтр по температуре — диапазон температур
6. График температуры — визуализация через matplotlib
7. Статистика — мин/макс/средняя температура
8. Выход

## Требования

- Python 3.6 или выше
- matplotlib (`pip install matplotlib`)
- Git
