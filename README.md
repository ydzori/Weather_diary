**Автор:** Ольшанская Дарья

**Вариант:** №1 Weather Diary

**Дата сдачи:** 19.06.2026

## Описание

Weather Diary - приложение для записи информации о погоде и построения графиков температуры.  
Позволяет добавлять, просматривать и удалять записи о погоде, фильтровать данные и визуализировать изменения температуры.

## Использование

Приложение предоставляет меню с 8 опциями:

1. Показать все записи
2. Добавить запись
3. Удалить запись
4. Фильтр по дате
5. Фильтр по температуре
6. График температуры
7. Статистика
8. Выход

## Требования

- Python 3.6 или выше
- matplotlib
- Git
